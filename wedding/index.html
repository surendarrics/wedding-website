Step-by-Step Guide

How to Create It

Follow these 8 simple steps to generate a premium animated wedding website using Claude AI — completely free.

01
🌐
Open Claude AI
Go to claude.ai and sign in or create a free account. Use the free tier — Claude Sonnet works perfectly for this.
claude.ai
02
🎯
Choose Your Theme
Decide the wedding style — South Indian / Hindu, Christian, or Islamic / Nikah. Each has unique architecture, animations, and cultural elements.
Pick a Style
03
📋
Copy the Prompt
Scroll to the Prompts section below. Select your theme and click Copy Prompt. This crafted instruction tells Claude exactly what to build.
Copy & Paste
04
🤖
Paste & Generate
Paste the prompt into Claude's chat and press Enter. Claude generates a complete single HTML file with all animations, SVGs, and features in about 30–60 seconds.
30–60 seconds
05
💾
Save the HTML File
Click Download on Claude's response, or copy the full code. Save as wedding.html. No server needed — opens in any browser.
Single HTML File
06
✏️
Customize Details
Open in any text editor. Search for EDIT HERE comments to update names, dates, venues, WhatsApp number, and photos instantly.
Ctrl+F → EDIT HERE
07
🖼️
Add Your Photos
Replace the placeholder image URLs with direct links to your photos. Host on Google Drive, Cloudinary, or Imgur — just replace the URL in the code.
Photo Upload
08
🚀
Share with Guests
Drag the HTML file to netlify.com/drop to get a free link instantly. Share it on WhatsApp — guests open it on any phone, beautifully.
netlify.com/drop
💡 PRO TIPS

Ask Claude to Modify

After generating, say: "Change the color palette to blush and ivory" or "Add a gallery section"

Mobile First

These are fully responsive. Test on your phone — most wedding guests open invitations on WhatsApp mobile.

Custom Domain

Netlify gives you a free URL like james-elizabeth.netlify.app — professional and shareable.

Add Your Music

Replace the audio src URL with a direct MP3 link. Host on Google Drive or any CDN and paste the URL in the code.

You are an expert full-stack web developer specializing in premium South Indian / Hindu wedding invitation websites.

Create a complete, single-page HTML wedding invitation website with:

HERO: Full-viewport South Indian temple SVG (Gopuram with tiers, kalasha finials, flagpoles, arched entrance, oil lamps/deepam). Starry night sky gradient from deep black to terracotta gold. Couple names in Cinzel Decorative font. Tamil/Sanskrit blessing at top.

FLOATING ELEMENTS: Scroll-triggered deepam lamps (SVG oil lamps) that descend as user scrolls — fixed positioning + JS scroll listener. Each lamp should sway and flicker using CSS keyframe animations.

SECTIONS (in order):
1. HERO — Temple SVG + stars + parallax text (names fade/move on scroll) + Tamil blessing
2. KOLAM DIVIDERS — SVG dot-pattern kolam dividers between sections
3. BLESSING — Sanskrit/Tamil verse, grandparents names, parents names, big couple names
4. EVENTS — Grid of invitation cards with clip-path corner cuts. Events: Mehendi/Nalangu, Haldi/Palichai, Sangeet, Kashi Yatra, Kalyanam (Wedding), Nichayathartham
5. COUPLE — Ornate corner-frame photo placeholder, individual bios with quote
6. RSVP — Dark maroon section, WhatsApp button
7. DETAILS — Grid cards: Hashtag, Weather, Hotel, Dress Code, Transport, Gifts
8. COUNTDOWN — Live JS countdown with stars background
9. FOOTER — Names, date, social links

TECH:
- Pure HTML5 + CSS3 + Vanilla JS (no frameworks)
- Google Fonts: Cinzel Decorative, Cinzel, Philosopher, Lora, Noto Serif Tamil
- Colors: deep red #3D0000, vermilion #C0392B, gold #C9942A, ivory #FDF6E3, cream #FAF0DC
- Scroll progress bar, IntersectionObserver reveals, parallax temple SVG, clip-path cards, music toggle, countdown

Output complete single HTML file.
